# Anticorruption AI Agent System

O'zbekiston qonunchiligini tahlil qiluvchi va korrupsion xavflarni topuvchi 6 agentli AI tizimi.

## Ishga tushirish (BITTA BOSISH)

Loyiha papkasidagi **`start.py`** faylini ikki marta bosing (double-click).

Shunda:
1. Server avtomatik ishga tushadi
2. Brauzer avtomatik ochiladi (`http://localhost:3000`)
3. Tayyor!

**Hech narsa o'rnatish shart emas.** Faqat kompyuterda Python bo'lsa yetarli.

## Saytdan foydalanish

1. Sayt ochilgach, yuqori o'ng burchakdagi **"Sozlamalar"** tugmasini bosing
2. Anthropic (Claude) API kalitingizni kiriting va Saqlash ni bosing
3. Qonun matnini yozing va **"Tahlilni Boshlash"** ni bosing
4. 6 ta AI agent qonunni tahlil qiladi va hisobotni chiqaradi

## Ma'lumotlar saqlash

- API kalit: `data/settings.json`
- Tahlil tarixi: `data/history.json`

Loyiha papkasini boshqa kompyuterga ko'chirsangiz, barcha sozlamalar va tarix saqlanib qoladi.
